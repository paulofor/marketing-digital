Guia de Uso - Prompt Estratégico Básico

Bem-vindo ao seu guia prático para utilizar o Prompt Estratégico Básico no ChatGPT. Siga o passo a passo abaixo para configurar o ambiente e utilizar o prompt com o máximo de eficácia.

Passo 1: Acessando o ChatGPT

  1. Entre no site do ChatGPT: Acesse o site ChatGPT e faça login na sua conta. Caso ainda não tenha uma, será necessário criar um cadastro.
  2. Escolha o modo de interação adequado: Existem diferentes versões do ChatGPT, como GPT-3.5 e GPT-4. Opte pela mais atual para garantir uma experiência de resposta mais precisa.

Passo 2: Preparando-se para Usar o Prompt

  1. Abra uma nova conversa: Comece com uma conversa limpa para que o ChatGPT não traga informações de interações anteriores.

  1. Copie o texto do Prompt Estratégico Básico: Encontra-se no quadro abaixo.
  2. Cole o prompt no ChatGPT: Cole o texto copiado no campo de conversa e envie.
